# Rigidbody
- 물리 시뮬레이션을 제어하는 컴포넌트
- 중력, 충돌, 힘 등의 물리 효과를 적용할 때 사용됩니다.
-- 오브젝트가 중력의 영향을 받게됨 Use Gravity
-- 중력 말고도 질량, 마찰력 등등 현실 세계에 존재하는 물리적인 힘들 설정 가능
-- Is Kinematic 가 체크가 되어 있다면 중력은 물론이고 모든 물리적인 힘들을 받지 않게 된다.

# 주의 사항
- FixedUpdate() 사용 : 물리 연산은 Update() 대신 FixedUpdate()에서 처리해야 안정적입니다.
- Kinematic = true 일때는 AddForce()가 작종하지 않습니다, MovePosition(), MoveRotation() 사용

# 기본 프로퍼티
- Mass : 질량
- Drag : 공기 저항
- Angular drag : 회전 저항
- Use Gravity : 중력 사용
- Is Kinematic : 물리 시뮬레이션을 무시하고 직접 제어할 때 (애니메이션 적용 오브젝트)


# 운동 상태 제어 프로퍼티
- 1. Velocity : 오브젝트의 현재 이동속도
-- rigidbody.velocity = new Vector3(0,0,5)

- 2. Angular Velocity : 회전속도
-- rigidbody.angularVelocity = Vector3.up * 2;

- 3. Max Angular Velocity : 회전 속도의 최대 값

# 충돌 및 제약 프로퍼티
- 1. Collision Detection : 충돌 검출 모드
-- Discrete : 기본값, 일반적인 충돌
-- Continuous : 고속 오브젝트용
-- Continuous Dynamic : 극고속 오브젝트용

- 2. Interpolate : 보간, 물리 움직임의 부드러움 조정
-- None
-- Interpolate : 이전 프레임 기반 보간
-- Extrapolate : 다음 프레임 예측

- 3. Constraints : 제약
-- Freeze Position/Rotation 으로 특정 축의 이동/회전 을 잠금


# 함수/메서드
##ForceMode
-Force : 지속적인힘
-Impuls : 순간적인 힘
-Acceleration : 징량 무시, 지속적인 가속도
-VelocityChange : 질량 무시, 순간적인 속도
- 1. AddForce(Vector3 force, ForceMode mode) : 오브젝트에 힘을 가합니다.
- 2. AddRelativeFoce(Vector3 force) : 로컬 좌표계 기준으로 힘들 가합니다. (차량의 전진/후진)
- 3. AddExplosionForce(float force, Vector3 position, float radius) : 특정위치에서 폭발력 적용(폭탄 효과)
- 4. AddTorque(Vector3 torque) : 오브젝트에 회전력을 가합니다.
- 5. AddRelativeTorque(Vector3 torque) : 로컬 좌표계 기준으로 회전력 적용
- 6. MovePosition(Vector3 position) : 물리 시물레이션을 우회하지 않고 부드럽게 이동
- 7. MoveRotation(Quaternion rotation) : 물리 시물레이션을 우회하지 않고 부드럽게 회전
- 8. Sleep() / WakeUp() : Rigidbody를 일시적으로 재우기, 깨우기
- 9. SetVelocity(Vector3 velocity) : 속도를 직접 설정
-- rigidbody.velocity = new Vector3(0, 10, 0)
- 10. SetAngularVelocity(Vector3 angularVelocity) : 각속도를 직접 설정
-- rigidbody.angularVelocity = new Vector3.up * 2;
- 11. sweepTest(Vector3 direction, out RaycastHit hit, float distance) : 특정 방향으로 충돌 검사
- 12. ClosesPointOnBounds(Vector3 position) : 오브젝트의 Collider 경계에서 가장 가까운 점 반환

# 이벤트/메시지
## 충돌 감지 메시지
사용조건 : Rigidbody가 있는 오브젝트 간 충돌
용도 : 벽에 충돌, 점프 착지 등
- 1. OnCollisionEnter(Collisition collision)
-- 처음 충돌이 시작될 때 한 번 호출됨
-- 총알이 변에 처음 맞았을 때 등등

- 2. OnCollisiionExit(Collision collision)
-- 충돌이 끝났을 때 호출됨

- 3. OnCollisionStay(Collision collision)
-- 충돌이 유지되는 동안 매 프레임마다 호출됨

## 트리거 감지 메시지
사용조건 : isTrigger 체크된 Collider
용도 : 아이템 줍기, 이벤트 트리거
- 1. OnTriggerEnter(Collider other)
-- isTrigger가 켜진 Collider에 처음 진입 했을 때 호출

- 2. OnTriggerExit(Collider other)
-- 트리거에서 벗어났을 때 호출됨

- 3. OnTriggerStay(Collider other)
-- 트리거 안에 있는 동안 반복 호출

## 물리 업데이트 메시지
- 1. OnBecameInvisible()
-- 오브젝트가 카메라 뷰에서 사라질때 호출됩니다.
