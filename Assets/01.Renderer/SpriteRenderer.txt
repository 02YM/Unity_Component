# SpriteRenderer
2D 스프라이트 이미지에 색상, 머티리얼, 레이어 등을 적용해 화면에 표시하는 컴포넌트

#기본/프로퍼티
- 1. Sprite : 어떤 스프라이트 이미지를 보여줄지 지정
- 2. Color : 스프라이트에 색상/알파 값을 입힝
- 3. Flip X/Y : 좌우 또는 상하 반전
- 4. Draw Mode : 단순렌더, 타일 형태, 슬라이스 등
- 5. Mask Interaction : 마스크 사용 야부 설정(UI/특수 효과용)
- 6. Sprite Sort Point : 스프라이트 정렬 포인트
- 7. Material : 셰이더를 적용해서 다양한 효과 가능
- 8. Sorting Layer : 렌더 순서 지정(다수 스프라이트 겹칠 때 필수)

# 변수/프로퍼티 Renderer.<T>
- 1. enabled
-- 오브젝트를 씬에서 보이게/숨기게 함

- 2. material
-- 오브젝트의 적용된 첫 번째 머티리얼을 가져옴
-- 수정하면 인스턴싱 처리됨

- 3. sharedMaterial
-- 공유 머티리얼 접근
-- 에디터 편집이나 최적화에서 사용

- 4. bounds
-- 월드 공간에서의 경계 박스 정보
-- 오브젝트의 크기나 위치를 시각적으로 처리할 때 사용
